# Test-ib description

Написать docker-compose.yml файл в котором будут подниматься виртуальные X, где будет работать хром (или другой браузер).  


Доступ к хрому должен осуществляться посредством vnc, порт которого должен быть доступен сразу после запуска контейнера/ов (порт может быть динамическим). 


Внутри контейнера браузер должен работать через tor.


В качестве результата принимается docker-compose.yml файл, который достаточно запустить  через  docker-compose up -d без каких  либо дополнительных настроек
